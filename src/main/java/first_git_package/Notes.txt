1.create a project
2.create a github repository
  *go to github
  *repositories>create new repository>first_git_project

3.create a first commit for the first time
  *git init
  *git add .
  *git commit -m "first commit"
  *git branch -M main
  *git remote add origin https://github.com/gizemguvel/first_git_project.git--link our local and github
  *git push -u origin main
  ***then your code should be pushed to github. refresh your page to see the code



NOTE: git status --> it can be used any time to see the progress. used to understand the status(where you are)

4.Create a second commit
  *create a new class: MySecondClass
  *create a commit and push the code to github
     *git add.
     *git commit -m "added my second java class"
     *git push
5.Adding specific classes or files
  *create 2 classes: MyThirdClass, MyFourthClass
  *create a commit with only MyFourthClass
    *git add "path"
    *git commit -m "adding fourth class"
    *git push
6.git pull
  *used to get code from github
  *git pull=git fetch + git merge
  *if git pull gives error use git fetch+git merge
  *now that we have our local updated, then commit and push
  *git add.
  *git commit -m "committing my codes"
  *git push
